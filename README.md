# ZCode Account Manager

GUI для управления несколькими аккаунтами ZCode (start-plan, OAuth Z.ai).

## Возможности
- **Текущий аккаунт** — показывает, какой аккаунт активен прямо сейчас
  (по `user_id` из JWT провайдера `builtin:zai-start-plan`).
- **Добавление** — сохраняет текущий залогиненный аккаунт как профиль.
- **Переключение** в один клик → закрывает ZCode, подменяет файлы, запускает заново.
- **Косметика** — поле «Подписка до», «Лимит/день», «Заметки» (только для вас, никуда не отправляются).

## Запуск
```cmd
python account_manager.py
```
Или двойным кликом по `run.cmd`.

## Как это работает
«Аккаунт» в ZCode (для start-plan) — это два файла:
- `~/.zcode/v2/credentials.json` — OAuth-токены (access_token, jwt, user_info)
- `~/.zcode/v2/config.json` — здесь в `provider["builtin:zai-start-plan"].options.apiKey`
  лежит JWT, в payload которого есть `user_id`. Это стабильный идентификатор аккаунта.

Менеджер:
1. Копирует эти два файла в папку профиля (`data/profiles/<id>/`).
2. При переключении — делает бэкап живых файлов в `data/backups/<timestamp>/`,
   затем копирует файлы выбранного профиля поверх живых.
3. Закрывает ZCode (`taskkill /F /IM ZCode.exe /T`) и запускает заново.

## Структура
```
zcode-account-manager/
├── account_manager.py     ← само приложение
├── run.cmd                ← запуск двойным кликом
├── README.md
└── data/
    ├── profiles.json      ← список профилей + косметика
    ├── profiles/<id>/     ← credentials.json + config.json каждого аккаунта
    └── backups/<ts>/      ← резервные копии перед каждым переключением
```

## Важно
- Работает **только** с провайдером `builtin:zai-start-plan`.
- Все файлы хранятся **локально**. Токены лежат в `data/profiles/` — это чувствительные данные, не коммитьте их никуда.
- Перед переключением текущий профиль обновляется (свежий access_token). Если текущий аккаунт не сохранён в менеджере — будет предупреждение (его логин потеряется).
- Бэкапы перед каждым переключением в `data/backups/` — откатиться можно вручную.

## Добавить 5 аккаунтов
Войдите в ZCode под первым аккаунтом → откройте менеджер → «Добавить текущий аккаунт».
Выйдите из ZCode, войдите под вторым → добавьте. Повторите для каждого.
Менеджер различает их по `user_id`, так что добавлять можно сколько угодно.
